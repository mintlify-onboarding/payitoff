#### Schema Definition

| Property | Type   | Format                          | Description                                |
| -------- | ------ | ------------------------------- | ------------------------------------------ |
| address  | object | [Address](/docs/address-schema) | Servicer's address.                        |
| id       | string | uuid                            | A unique UUID identifier for this Servicer |
| name     | string | Servicer's name.                |                                            |
