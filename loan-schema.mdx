A `Loan` object provides the details fo a single student loan for a `Borrower`.

> When creating `Loans` for `Borrowers` via the [Create Loans](/reference/create-loans) API endpoint, you must supply loan details in the shape of a [LoanPayload](/docs/loan-payload-schema).
> 
> You will receive a `Loan` object in the API response.
