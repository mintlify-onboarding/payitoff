description: "Payitoff API: RepaymentOptions Schema"
---

When you query our [Get Repayment Options](/reference/get-repayment-options) API endpoint, you'll receive a JSON response object with an `array` of [RepaymentOption](/docs/repayment-option-schema) objects, alongside an [OriginalLoanSummary](/docs/original-loan-summary-schema) 

#### Schema Definition

| Property           | Type   | Format                                                    | Description                                                                                                            |
| ------------------ | ------ | --------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| original           | object | [OriginalLoanSummary](/docs/original-loan-summary-schema) | Summary of original loan details against which you'd compare RepaymentOption objects for which a Borrower is eligible. |
| repayment\_options | array  | [RepaymentOption](/docs/repayment-option-schema)          | An array of available RepaymentOption objects for which a Borrower and their Loans may be eligible.                    |
